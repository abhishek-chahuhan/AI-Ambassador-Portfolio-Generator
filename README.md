# AI Ambassador Portfolio Generator

## 📌 Overview

AI Ambassador Portfolio Generator is a web application that helps Google Student Ambassadors create professional LinkedIn posters and captions using AI.

Instead of spending hours designing posts manually, users can enter their monthly activities, upload a photo (optional), and generate beautiful, ready-to-share posters.

---

## ✨ Features

- Upload profile photo (Optional)
- Enter Name & GID
- Select Monthly Highlights
- AI-powered content enhancement
- Professional poster generation
- LinkedIn caption generation
- Download poster as PNG
- Multiple templates
